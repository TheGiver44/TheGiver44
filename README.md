### 🦊 [ZAP Wallet Chrome Extension](https://zapzip.fun/)
> **The First AI Wallet on Solana** <br>
> *The fun way to create and manage Solana wallets. Accessible to everyone with AI-powered ideas and a colorful, engaging interface.*

<a href="https://zapzip.fun/"><img src="https://img.shields.io/badge/Live-zapzip.fun-7C3AED?style=flat-square&logo=googlechrome&logoColor=white" /></a> <a href="https://x.com/zapzipwallet"><img src="https://img.shields.io/badge/X-@zapzipwallet-000000?style=flat-square&logo=x&logoColor=white" /></a>

- **Core Features:** BIP44 compatible generation, local Chrome storage, Export/Import, Phantom compatibility, fully client-side (No Backend Required).
- **Coming Soon:** Send/Receive SOL via QR, Token Swaps, DeFi Integrations.

<br>

### 🪞 Mirror
> **Advanced Solana Trading & Buyback Management** <br>
> *A Next.js application for lightning-fast Solana trading with intelligent buyback policies, powered by PumpPortal integration.*

- **Features:** Millisecond execution via PumpPortal API, local trading with treasury integration, automated buyback strategies, Redis-based idempotency locks, real-time monitoring, and a modern Tailwind UI.

<br>
